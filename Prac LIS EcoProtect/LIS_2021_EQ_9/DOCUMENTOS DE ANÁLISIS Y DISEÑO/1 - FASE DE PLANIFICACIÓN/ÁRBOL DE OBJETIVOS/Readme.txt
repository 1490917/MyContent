Para la definición de un proyecto es vital realizar previamente un análisis en el que se identifique: 
-Cuál es la motivación del proyecto
-Concienciación de la problemática a resolver
-Acotar el propósito del proyecto
-Identificación del impacto del proyecto a corto y largo plazo

Para la identificación de objetivos del proyecto se ha utilizado el método de árbol Medios-Fines.
Un sistema que permite ordenar los objetivos de manera jerárquica.

Primeramente se ha construido el árbol del problema:
-define el problema central
-recoge los efectos del problema central
-determina las causas del problema central

Seguidamente se ha dibujado el árbol de medios fines, que transforma
-Causas ---> Medios
-Efectos ---> Fines

Finalmente se ha refinado este último árbol, usando el patrón sintáctico (verbo infinitivo + complementos).
